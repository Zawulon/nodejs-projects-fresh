1% podatku dla fundacji

Program PIT 2017 jest DARMOWY dla wszystkich podatników.

Cele Fundacji są realizowane dzięki wsparciu wszystkich życzliwych osób, ogromnemu zaangażowaniu wolontariuszy jak również finansowemu wsparciu Firm – rozumiejących potrzeby innych.
Dzięki darczyńcom jednego procenta wspierającym działalność Fundacji możemy pomóc innym

Nasz nr KRS:0000050483

Program nr 1 – PIT 2017 Format

Cechy programu PIT Format 2017:
• zawiera 8 typów deklaracji podatkowych (PIT-28, PIT-28A, PIT-28B, PIT-36, PIT-36L, PIT-37, PIT-38, PIT-39 ), załączniki (PIT-B, PIT-D, PIT-O, PIT-2K, PIT-M, PIT-ZG, PIT-Z) oraz formularze NIP (NIP-2, NIP-7, NIP-B, NIP-C, NIP-D, NIP 2A, ZAP 3)
• sprawnie wysyła e-deklaracje do systemu ministerstwa lub drukuje roczne rozliczenia
• posiada Kreator z przyjaznym systemem „krok po kroku”
• generuje wydruki, co ważne dane wpisane do jednego druku można eksportować do innego, dzięki czemu nie ma konieczności wprowa

dzania wielokrotnie tych samych danych
• daje możliwość zapisu deklaracji do pliku PDF
• jest stale aktualizowany z przepisami podatkowymi
• ma wbudowaną wewnętrzną bazę danych: m.in. lista urzędów skarbowych, miejscowości, województw, a także zapamiętuje podstawowe dane podatnika!
• zawiera obszerną dokumentację, zbiór przepisów oraz wbudowane mechanizmy kontroli i kontekstowych podpowiedzi!
• w każdym miejscu programu dostępny podręczny kalkulator walut!

Program PIT 2017 Format przeznaczony jest dla wszystkich, którzy muszą wypełnić PITy 2017 w 2018 roku.

Łatwy i szybki w obsłudze program sprawi, że rozliczenie pit nie będzie już czarną magią! Z programem wypełnisz pita sprawnie i bezbłędnie.

Co sprawia, że program pity 2017 jest tak chętnie stosowany do rozliczenia podatku?
• intuicyjna i prosta obsługa programu – deklarację wypełnia się nawet w 5 minut
• program pity 2017 zawiera komplet deklaracji podatkowych i załączników tak, by każdy rozliczył się fiskusem
• program dostępny w dwóch wersjach: instalacyjnej oraz internetowej (nowość!), dzięki czemu z programu mogą korzystać internauci i użytkownicy systemów: Windows, Mac OS, Linux
• możliwość wyboru drogi sporządzenia zeznania podatkowego – aktywne deklaracje oraz kreator prowadzący „krok po kroku”
• sprawna wysyłka e-deklaracji
• system e-deklaracji rozbudowany jest o obsługę najczęstszych problemów przy wysyłce elektronicznej
• program PIT jest jednym z najczęściej pobieranych aplikacji na onet.pl
• automatyczna, bezpieczna aktualizacja programu – program jest zawsze zgodny z obowiązującymi przepisami

Wykonawcą programu PIT FORMAT jest Biuro Informatyki Stosowanej FORMAT –   www.programy-format.pl
